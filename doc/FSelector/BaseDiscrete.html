Overview

base alogrithm for handling discrete feature

2 x 2 contingency table

      c   c'
    ---------
 f  | A | B | A+B
    |---|---| 
 f' | C | D | C+D
    ---------
     A+C B+D  N = A+B+C+D

 P(f)     = (A+B)/N
 P(f')    = (C+D)/N
 P(c)     = (A+C)/N
 P(c')    = (B+D)/N
 P(f,c)   = A/N
 P(f,c')  = B/N
 P(f',c)  = C/N
 P(f',c') = D/N
 P(f|c)   = A/(A+C)
 P(f|c')  = B/(B+D)
 P(f'|c)  = C/(A+C)
 P(f'|c') = D/(B+D)
